Quy tắc tuyến tính: $\int[f+g]\,dx = \int f\,dx + \int g\,dx$.

Dạng $ax+b$ trong căn hoặc mũ: đặt $u=ax+b$, $du=a\,dx$.

Dạng $f'(x)/f(x)$: kết quả $= \ln|f(x)|+C$.

Dạng $f'(x)\cdot e^{f(x)}$: kết quả $= e^{f(x)}+C$.

Tích phân từng phần: $\int u\,dv = uv - \int v\,du$. Dùng khi có $x\cdot e^x$, $x\cdot\sin x$,...

Tính tích phân xác định: Bấm SHIFTF3 (hoặc MENU → Integral): nhập hàm, cận dưới, cận trên.

Ví dụ: $\int_0^1 x^2\,dx = 1/3$: Bấm integral(X^2, 0, 1) → 0.333...

Kiểm tra nguyên hàm: Tính $F(b)-F(a)$ bằng tay rồi so với máy tính tích phân số.

Lưu ý: Máy cho kết quả số thập phân; đối chiếu với kết quả phân số tính tay.

Bài 1: Tính $\displaystyle\int_0^2 (3x^2-2x+1)\,dx$.

$= \left[x^3-x^2+x\right]_0^2 = (8-4+2) - 0 = 6$.

Bài 2: Tính $\displaystyle\int_0^1 \frac{2x}{x^2+1}\,dx$.

Dạng $f'(x)/f(x)$ với $f(x)=x^2+1$, $f'(x)=2x$.

$= \left[\ln(x^2+1)\right]_0^1 = \ln 2 - \ln 1 = \ln 2 \approx 0.693$.

Ứng dụng tích phân

Diện tích giữa 2 đường cong: $S = \int_a^b |f(x)-g(x)|\,dx$.

Tìm dấu trước: vẽ sơ đồ hoặc tính tại điểm mẫu để biết hàm nào lớn hơn.

Thể tích khối tròn xoay quanh Ox: $V = \pi\int_a^b [f(x)]^2\,dx$.

Thể tích quanh Oy: $V = \pi\int_c^d [g(y)]^2\,dy$.

Diện tích giữa 2 hàm: Tính integral(|f(x)-g(x)|, a, b) – máy tự tính giá trị tuyệt đối.

Thể tích: Tính pi * integral(f(x)^2, a, b) – nhập $\pi$ × tích phân.

Nếu khó tính chính xác: Dùng tích phân số để kiểm tra đáp án.

Bài 1: Tính diện tích hình phẳng giới hạn bởi $y=x^2$ và $y=x$.

Giao điểm: $x^2=x \Rightarrow x=0, x=1$. Trên $[0,1]$: $x > x^2$.

$S = \int_0^1(x-x^2)\,dx = \left[\frac{x^2}{2}-\frac{x^3}{3}\right]_0^1 = \frac{1}{2}-\frac{1}{3} = \frac{1}{6}$.

Bài 2: Tính thể tích vật thể tròn xoay khi quay hình phẳng giới hạn bởi $y=\sqrt{x}$, trục Ox, $x=4$ quanh Ox.

$V = \pi\int_0^4 (\sqrt{x})^2\,dx = \pi\int_0^4 x\,dx = \pi\left[\frac{x^2}{2}\right]_0^4 = \pi \cdot 8 = 8\pi$.
